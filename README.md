## 📖 Overview

A personal blog and portfolio site built with Next.js 16 App Router. Features a clean, responsive design with dynamic blog posts, project showcase, and admin dashboard.

## ✨ Features

- **MDX Blog** — Write posts in Markdown with React components
- **Project Showcase** — Portfolio page with project cards
- **Admin Dashboard** — CRUD for posts and projects, protected by authentication
- **Dark Mode** — next-themes with system preference detection
- **Responsive** — Mobile-first design with Tailwind CSS
- **Reading Time** — Automatic reading time estimation

## 🚀 Quick Start

```bash
npm install
npm run dev
# Open http://localhost:3000
```

## 🔧 Tech Stack

- **Framework**: Next.js 16 (App Router)
- **UI**: React 19, Tailwind CSS, Framer Motion, Radix UI
- **Database**: better-sqlite3
- **Auth**: Custom credential-based auth
- **Content**: MDX with syntax highlighting
